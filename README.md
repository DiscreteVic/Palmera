# Palmera
Palmera tool for device trees managing, build completely  on Rust.


Basic usage:
```
palmera -i <input binary file> -o <output dtb file>
```
## Version 0.2.0
-   Binary inspection,  device tree detection.
-   Device tree extraction.

### Next features
- Instalation script.
- Binary injection.
- Output file format election, dts or dtb.
- Device tree live editor.
